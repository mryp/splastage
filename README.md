# splastage
スプラトゥーンのステージ情報を取得し管理するWebアプリケーション

## Splastage

スプラトゥーンのステージ情報をイカリングから収集し配信用APIを提供するサーバープログラムです。
なお、Go言語で作成されています。

## ライブラリ

下記のライブラリを使用します。

Revel（Webフレームワーク）
https://revel.github.io/

Gorp（SQL操作）
https://github.com/go-gorp/gorp

goquery（HTMLパーサ）
https://github.com/PuerkitoBio/goquery
